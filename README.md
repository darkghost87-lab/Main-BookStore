## 📖 درباره‌ی پروژه

**BookStore** یک پروژه‌ی **فرانت‌اند** برای فروشگاه اینترنتی کتاب است که با هدف ارائه‌ی تجربه‌ای دلنشین و جذاب برای علاقه‌مندان به کتاب طراحی شده است. از انتخاب رنگ‌های گرم و دل‌نشین گرفته تا چیدمان منظم و روان‌سازی تعاملات، همه‌چیز با دقت بالا پیاده‌سازی شده تا کاربر حس خوبی در حین جستجو و مرور کتاب‌ها داشته باشد.

> ⚠️ **توجه:** این پروژه در حال حاضر صرفاً در سمت کاربر (فرانت‌اند) توسعه‌یافته و هنوز به بک‌اند متصل نیست. داده‌های کتاب‌ها به‌صورت شبیه‌سازی‌شده نمایش داده می‌شوند.

## 🚀 نصب و اجرا

برای اجرای پروژه روی سیستم خود، این مراحل را دنبال کنید:

```bash
# 1. کلون کردن مخزن
git clone https://github.com/darkghost87-lab/Main-BookStore.git

# 2. ورود به پوشه‌ی پروژه
cd BookStore_main

# 3. اجرا با Live Server (پیشنهادی)
# اگر از VS Code استفاده می‌کنید، روی فایل index.html راست‌کلیک کرده و "Open with Live Server" را انتخاب کنید.
# 💡 نکته: برای بهترین تجربه، پروژه را با یک سرور محلی اجرا کنید.
```
📁 ساختار پروژه
BookStore_main/
├── index.html          # فایل اصلی HTML
├── style.css           # استایل‌های پروژه
├── script.js           # منطق جاوااسکریپت
└── img/                # تصاویر و آیکون‌های کتاب‌ها
